#### tree

```bash
cd 目标文件夹路径
```

然后 tree 一下

```undefined
tree
```

选项

| 选项       | 含义                                                   |
| ---------- | ------------------------------------------------------ |
| -a         | 显示所有文件和目录                                     |
| -d         | 只显示目录名称，不显示文件                             |
| -D         | 列出文件或目录的更改时间                               |
| -L num     | 显示num层目录结构，深度大禹num层的目录和文件将不会显示 |
| - I        | 过滤文件或文件夹                                       |
| - e '文件' | 输出到某文件                                           |

tree -I pattern 用于过滤不想要显示的文件或者文件夹。比如要过滤项目中的node_modules文件夹：

```
tree -I “node_modules”
```
